四条腿好，两条腿坏 ——《动物农场》

发表于 2016-12-18 | 更新于 2017-01-09 | 分类于 读书 | | 阅读次数：
  • 书名：《动物农场》
  • 作者：[英]乔治·奥威尔（1903-1950）
  • 时间：2016年11月

第一章 动物集会

一天，农场里一只年迈的猪老少校晚上做了一个奇怪的梦，然后传达给其他动物。它在提出“人类剥削牲畜，牲畜须革命”的理论之后死去。

第二章 反抗成功

老少校的思想在动物们之间传开来，几头猪拿破仑、斯诺鲍、斯奎拉也在发展动物主义。琼斯先生也在动物们势如破竹的追赶下仓皇败逃，赶走人类后，他们建立了自己的组织，提出了七诫。

七诫

  1. 凡靠两条腿行走者皆为仇敌；
  2. 凡靠四肢行走者，或者长翅膀者，皆为亲友；
  3. 任何动物不得着衣；
  4. 任何动物不得卧床；
  5. 任何动物不得饮酒；
  6. 任何动物不得伤害其他动物；
  7. 所有动物一律平等。

第三章 猪的作用

猪非常聪明，他们阅读书写学习各种技能，并不干活，只是指导和监督其他动物，成为独立后的动物农场的领导者。

  • 像羊、鸡、鸭这些比较笨的动物，没法儿把《七诫》全记住。经过深思熟虑，雪球宣布《七诫》其实可以压缩为一句格言：“四条腿好 ，两条腿坏。”他说，这句格言包括了动物主义的精髓实质。
  • 禽类的翅膀是主要起推进作用的器官，而不是主要起操控作用的器官，所以应当被看作是腿。人的区别性标志是手，人正是用它来干一切坏事的。
  • 有一天，发布了这样一个指示，说是让把所有被风吹落下来的苹果收集起来，带到农具室去供猪食用。

    “同志们，你们不会把我们猪这样做看成是出于自私和特权吧？我希望你们不。实际上，我们中有许多猪根本不喜欢牛奶和苹果。我自己就很不喜欢。我们食用这些东西的唯一目的是要保护我们的健康。牛奶和苹果（这一点已经被科学所证明，同志们）包含的营养对猪的健康来说是绝对必需的。我们猪是脑力劳动者。庄园的全部管理和组织工作都要依靠我们。我们夜以继日地为大家的幸福费尽心机。因此，这是为了你们，我们才喝牛奶，才吃苹果的。你们知道吧，万一我们猪失职了，那会发生什么事情呢？琼斯会卷土重来！是的，琼斯会卷土重来！真的，同志们！”

第四章 牛棚战役

琼斯先生带领他的伙计发起反攻，幸好雪球研读了征战的旧书，动物们有力地挫败了琼斯先生的反攻。由于作战英勇，雪球被授予了“一级动物英雄”；

第五章 风车计划

作为领导者之一的雪球提出要在农场制高点建一座风车来发电，通过电力使动物们彻底摆脱繁重的劳动。为此，他通过研读农场里留下相关书籍并最终设计了一套完整的建设方案。另一位领导者拿破仑反对雪球提出的所有主张，自然也包括风车计划。在最后全民投票表决的时候，拿破仑利用狗发动政变赶走了雪球。随后他便宣称风车计划完全是由他设计的，雪球只是剽窃了他的成果，因而风车计划要继续实施下去。

“ 同志们 ” 他说， “ 拿破仑同志挑起了这副额外的重担，我相信这里的每一只动物都高度评价他所做出的牺牲。同志们，别以为当领袖是件开心事儿！相反，这是一份深层次、沉甸甸的职
责。没有谁比拿破仑同志更坚定地相信所有动物一律平等。他巴不得能让你们自己为自己做出决定。但有时候你们可能会做出错误的决定，同志们，那时我们将陷于何种境地？试想，假如你们决定跟着雪球走，去做他的风车白日梦一一那么，雪球，这个据我们现在所知比罪犯好不到哪儿去的雪球……”

第六章 功亏一篑

动物们开始建风车，而同时他们又要完成农场的既定任务，任务繁重。可就当风车初步建成的时候，一场暴风雨将一切化为乌有。

  • 一个星期天上午，动物们聚集到一起接受任务。拿破仑宣布 ，他决定实行一项新政策。从今往后，动物农场将同附近别的农场进行贸易往来。当然不是为了达到任何商业上的目的，而只
    是为了获得某些迫切需要的材料。他说，建造风车的需要必须压倒其他一切需要。为此他打算卖掉一垛干草和部分当年的小麦收成，以后如果还需要花钱，就得靠卖鸡蛋弥补缺口，反正鸡蛋在维林敦一直有销路。拿破仑说，母鸡应当愉快地承受这样的牺牲，作为她们自己对建造风车的贡献。
  • 大概也就在那时候，猪们一下子搬进了农场主的住宅，把那里作为他们的宿舍楼。
  • 他们听说猪们不单单在厨房里用餐，并把起居室当作娱乐室，而且还睡在床上。
  • 于是，第四条戒律变成了：凡动物都不可睡床铺被单
  • “同志们，看来，”他指出， “你们已经听说，我们猪现在睡到农场主住宅的床上去了，是不是？干吗不睡？莫非你们以为什么时候有这一条针对睡床的禁令不成？床的意思仅仅是睡觉的地方而已。从正确的观点来看，圈棚里的一堆干草同样是床。戒律针对的是被单， 因为那是人类的发明。我们已经把被单从农场主宅内的床上撤去，睡在上下两条毯子中间。那也是非常舒适的床铺！但是 ，我可以告诉你们，同志们，考虑到眼下有那么多伤脑筋的工作都得由我们去做，这还够不上我们所需要的舒适程度。你们不至于想要剥夺我们休息的权利吧，同志们？难道你们要把我们累得没法履行我们的职责不成？你们中不会有谁愿意看到琼斯回来吧?”
  • 过了几天，总部宣布从今以后猪每天早晨要比其他动物晚一小时起床，通知下达时，同样没有谁发牢骚。
  • “同志们，”他胸有成竹地说， “你们可知道，应该对此负责的是谁？你们可知道，是谁夜里摸黑进来把我们的风车搞塌了？是雪球!”他突然发出声如霹雳的咆哮。 “是雪球干下了这件事！这个叛徒纯粹出于险恶的用心，妄图使我们的计划开倒车，为他自己可耻地遭到驱逐进行报复，于是在夜幕的掩盖下潜入此地把我们将近一年的劳动成果毁于一旦。同志们，我在此时此地宣布判处雪球死刑。任何动物如能将雪球绳之以法，将 被 授 予 ‘动物英雄二等勋章’，还可得到十八升苹果。任何动物要是把他活捉了，可得到三十六升苹果!”

第七章 拨乱反正

一月份粮食告急，于是拿破仑要求母鸡交出所有下的蛋，愤怒的母鸡们进行了一次反抗，最终以死掉九只母鸡的代价而宣告失败。所谓的雪球暗中活动“迫使”拿破仑展开了全面调查，最终在一次会议上进行了全面清洗，多只承认与雪球有勾结的动物遭到处决。

遵照拿破仑同志的一项特别法令， 《英格兰的生灵》 已被取缔。从今以后，这首歌不准再唱。

“《英格兰的生灵》是造反之歌。但造反现已完成。今天下午处决一批叛徒是最后一幕。外部和内部的敌对分子都已被打败。过去我们通过《英格兰的生灵》表达的是对于未来一个更美好社会的渴望。但这个社会现在已经建成。很明显，这首歌不再有任何用途。”

第八章 风车战役

  • 拿破仑变成了“我们的领袖拿破仑同志”，其待遇和安全保卫又提升了一个等级。

  • 为了筹措购买风车配套设备的资金，拿破仑打算将农场的一部分木材卖给临近的一个农场主弗雷德里克，而他先与另外一个农场主皮尔金顿交好，迫使弗雷德里克将报价提高了十二磅，并最终以五磅面额的现钞支付。成交后，动物们被安排排成一行一一瞻仰这笔现金。不幸的是，三天之后，拿破仑收到中间人温珀的消息，钱是假的。拿破仑震怒，当众宣布判处了弗雷德里克死刑，而更糟糕的是不久后弗雷德里克率领手下共十五个人，六条枪成功攻入农场，并在众目睽睽下用炸药炸掉了风车。这一下子激怒了动物们，他们个个英勇冲锋，成功将入侵者赶了出去。风车战役后，拿破仑为牺牲的动物举行了隆重的葬礼，并进行了两天的庆祝胜利活动，拿破仑又获得了一枚勋章，假钞事件已被忘却。

  • 猪儿们在酒窖发现了一箱威士忌，当晚开箱饮酒。第二天早上，动物们被告知拿破仑同志病危，大家都表示非常担心，到了晚上病情好转，次日上午康复，当晚就又恢复了工作。不久后，拿破仑就购买了酿造和蒸馏的册子，并在原本打算留给退休动物的一片小围场上种上了大麦。

  • 任何动物不得 无故 杀害其他动物
  • 你时常可以听到一只母鸡在告诉另一只母鸡：“在我们的领袖拿破仑同志指引下， 我在六天里头产了五个蛋” ，或者两头母牛在池边饮水时会赞叹：“感谢拿破仑同志领导有方，这水的味道真是好极了! ”
  • 任何动物不得 过量 饮酒

第九章 拳师之死

最忠诚且任劳任怨的拳师病倒了，拿破仑同志找人将他送到了城里的医院，不久拳师就在医院里去世了。而接拳师去医院的马车上刻有这样的文字：

阿尔弗雷德·西蒙玆，屠马兼熬胶，住维林敦镇。经销兽皮和骨粉。可为养犬客户送货上门。
尽管有动物认出了文字，但是他们最终还是选择相信了上面传达给他们的事实：拳师在医院里获得了精心照料，但很不幸医治无效去世了。

第十章 人兽之疑

  • 几年过去了，很多寿命不长的动物已经去世，能够记得造反前是怎么回事的只剩下苜蓿、本杰明、乌鸦摩西和几头猪。农场似乎比往昔富裕了，面积扩大了，生产组织得也比较好，动物增加了不少，还从别处购进了一些动物。风车最终建成了，用于研磨谷物，第二座风车也在建设中，这座风车将安装发电机组。

  • 猪儿们学会了用后腿走路，并穿上了衣服，开始购置收音机，订阅杂志，准备安装电话。一周后的下午，附近几位农场主组成的代表团应邀前来考察，当晚，猪儿们举行晚宴进行招待。晚宴上，农场主们对猪儿们对农场的管理表示了认可和赞赏，而拿破仑正式宣布将“动物农场”这个名称重新改为“庄园农场”，一头猪成功实现了向人的蜕变。

  • 猪不得不每天耗费大量劳动在叫做“档案”、“报告”、“议事录”、 “备忘录” 的神秘事务上头。那都是大张大张的
    纸 ，必须在上面密密麻麻地写上字，一旦这些纸写满了字，就会放到炉子里烧掉。这对农场的福祉都是至关重要的，
  • 至于其他动物，据他们所知，他们的生活还一直是老样子。他们普遍吃不饱，睡干草，喝池塘水，干农活；冬天他们苦于寒冷，夏天受苍蝇骚扰。
  • 当时的日子是比现在好，还是比现在差。他们记不起来了。他们没有任何东西可以拿来同当前的生活做
    比较，因为他们没有任何参照的依据，除非以吱嘎的长长一大串一大串数字为准，而这些数字一贯表明任何事物都是越来越好，越来越好。
  • 那是一头猪在用他的后腿行走。

    没错，那是吱嘎。

    他正从院子的一头向另一头踱去，稍稍显得有点儿笨拙，仿佛还不太习惯按这样的姿势支撑自己硕大的身躯 ，但平衡保持得十分完美。仅仅过了片刻，从农场主宅子门内走出一长列猪，全都用他们的两条后腿行走。 一 些猪走得比另一些较好 ，有一两头甚至步态略显不稳，看样子他们最好能有一根拐棍作支持，但他们每一头都成功地绕院子走了一圈。临了是一片惊心动魄的狗叫和黑色小公鸡喔喔喔的尖声啼鸣，于是拿破仓亲自驾临，气宇轩昂，目光傲慢地从这一边扫向另一边，他的狗保镖们在他周围又蹦又跳。

    他的前蹄夹着一条鞭子。

    这时出现了一片死一般的寂静
  • 四条腿好，两条腿更好
  • 如今墙上只有一条戒律，其余什么都没有。那唯一的一诫是：凡动物一律平等，但是有些动物比别的动物更加平等
  • 敢情动物们从窗外朝里望，目光从猪移到人，再从人移到猪，又重新从猪移到人，要分清哪张脸是猪的，哪张脸是人的，已经不可能了。
