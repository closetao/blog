智识训练营 | 02.咨询顾问的秘诀：结构化思维

发表于 2018-07-25 | 分类于 课程笔记 | | 阅读次数：

本节课提到的结构化思维包括一个原则，三个思维模型：

  • 金字塔原则
  • WYH模型
  • GHC模型
  • 5W1H模型

2.1 金字塔原则

概念说明

一个金字塔结构包括一个高层信息和若干个低层信息，低层信息之间是平级的，否则就需要重新将其拆分成金字塔嵌入其中（即：层级），低层信息解释或说明高层信息（即：支撑），若干个低层信息之间是有分组的，不同的组之间又一定的逻辑关系（即：分类）。

分类

  • 并列型金字塔；

    低层信息之间是并列关系

  • 递进型金字塔

    低层信息之间是递进关系

并列与递进型金字塔举例

使用

  • 阅读时，可以将信息可以拆分成一个嵌套结构的金字塔模型，以便于结构化理解。

  • 自下而上：提炼信息要点

    1. 框架化：划分信息层级
    2. 组块化和重构：提炼信息要点
    3. 寻找要点之间的逻辑关系
    4. 重构逻辑框架和文字表述

  • 写作时，则是一个相反的过程，先有一个金字塔结构的框架，然后从框架出发，有序展开并填充细节。

  • 自上而下：有效传达信息

    1. 结论先行
    2. 提炼要点，高度浓缩信息内容
    3. 自左向右，逢交叉点下钻，讲透信息点
    4. 下钻时，用疑问引导，提高[预知性]

2.2 三个思维模型

WYH模型

WYH模型，即：What，Why，How，该模型可以与概念图相结合，用于弄清知识脉络，

  • 阅读：用WYH提炼和整理信息：
    • 主题：对方表达了一个什么主要信息？
    • 原因：支持这个信息的主要论点是什么？
    • 操作：基于这个信息，对方提出了什么建议或操作？

  • 写作：用WYH规划写作内容：

    • 主题：我今天要讲什么内容？
    • 解释：为什么会这样？背后有什么原因？
    • 方法：基于这些原因，我们可以怎么做？

  • 做读书笔记：围绕一个知识点用WYH将其学透

    • 含义：这个知识点是什么意思？
    • 逻辑：这个知识点是怎么来的？为什么会这样？
    • 应用：我可以如何应用这个知识点？

GHC模型

GHC模型，即：Good, Harm, Conclusion，结合WYH模型，通常可以用于全面剖析一个观点。

5W1H模型

5W1H模型，即：What, Who, Why, When, Where, How，通常用于全面阐述一个事实。
